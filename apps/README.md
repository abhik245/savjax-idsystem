# NexID Enterprise Monorepo

This folder contains the next-generation SaaS implementation:

- `web`: Next.js App Router frontend (TypeScript, Tailwind, Framer Motion)
- `api`: NestJS backend (Prisma, PostgreSQL, Redis-ready auth/session modules)

Primary local URLs:

- `http://localhost:3000/login`
- `http://localhost:3000/dashboard`
- `http://localhost:3000/dashboard/access`
- `http://localhost:3000/parent/intake?intake_token=<TOKEN>`
- `http://localhost:3000/parent/portal`
